How it works

One command. No installation. Your terminal becomes a classroom.

1

Open your terminal

On Mac: Terminal or iTerm. On Windows: Windows Terminal with WSL.

2

Run one command

npx tuilder — that's it. No install needed.

3

Learn by doing

Each lesson explains a concept and asks you to practice it in a safe simulated environment.

4

Graduate

Complete all 65 lessons, earn your certificate, and start contributing to real projects.
